# Reproducibilidad (Postverdad en Redes)

## Versionado
- Repositorio principal con commit hash registrado en `metrics.json`.

## Entorno
- Python 3.10+
- Dependencias: `numpy`, `pandas`, `requests`.

## Sensibilidad
- Se reportan min/max de medias bajo perturbaciones de parametros.

## Datos
- Fuente Wikimedia Pageviews API, cache local en `data/posttruth_wikipedia_monthly.csv`.
