# Arquitectura (Justicia y Normatividad)

## Capas
- Conceptual: normas como regimenes institucionales con legitimidad variable.
- Formal: ABM de cumplimiento + ODE de legitimidad agregada.
- Computacional: simulacion discreta con acople macro.
- Validacion: C1-C5 con fase sintetica y real.

## Variables
- Micro: nivel de cumplimiento por agente/celda.
- Macro: indice de rule of law.
- Forcing: shocks externos (reformas, crisis).

## Supuestos
- El indice RL.EST es un proxy razonable de normatividad.
- La dinamica puede aproximarse como transicion de regimen.
