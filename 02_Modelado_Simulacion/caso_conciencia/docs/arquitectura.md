# Arquitectura (Conciencia Fenomenica)

## Capas
- Conceptual: experiencia subjetiva aproximada por proxies reportados.
- Formal: ABM de estados afectivos + ODE de satisfaccion agregada.
- Computacional: simulacion discreta con acople macro.
- Validacion: C1-C5 con fase sintetica y real.

## Variables
- Micro: estado afectivo por agente/celda.
- Macro: nivel de satisfaccion promedio (Cantril ladder).
- Forcing: shocks externos (crisis, recuperacion).

## Supuestos
- La experiencia fenomenica no es observable directamente.
- El proxy (satisfaccion reportada) captura tendencia agregada, no esencia ontologica.
