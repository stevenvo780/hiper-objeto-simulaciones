# Protocolo (Politicas Estrategicas)

1. Cargar datos reales (WGI Regulatory Quality) y generar fase sintetica.
2. Normalizar observaciones (z-score).
3. Construir forcing con componente adversarial dependiente del estado.
4. Calibrar parametros y evaluar validacion.
5. Reportar C1-C5 e indicadores.
